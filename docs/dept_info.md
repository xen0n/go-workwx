# 部门管理

## Models

### `DeptInfo` 部门信息

Name| JSON       |Type|Doc
:---|:-----------|:---|:--
`ID`| `id`       |`int64`|部门 ID
`Name`| `name`     |`string`|部门名称
`NameEn`| `name_en`  |`string`|部门英文名称
`ParentID`| `parentid` |`int64`|父亲部门id。根部门为1
`Order`| `order`    |`uint32`|在父部门中的次序值。order值大的排序靠前。值范围是[0, 2^32)
